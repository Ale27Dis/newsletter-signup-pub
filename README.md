<h1 align="center">Newletter Signup Web App</h1>
<p>A Newsletter Signup Web app built using NodeJs and Mailchimp Marketing API.</p>

<h2>Requirements</h2>
<ul>
  <li>Node Js</li>
    <ul>Modules Used:
      <li>Express Js</li>
      <li>Mailchimp Client Library</li>
      <li>Body Parser</li>
  </ul>
  <li>Mailchimp API</li>
  <li>Heroku (for Deployment)</li>
</ul>
